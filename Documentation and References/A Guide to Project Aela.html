Introduction to Project Aela

What is Project Aela?

Project Aela is a software engine that specializes in giving a developer the ability to create any kind of desktop application with ease. Originally intended for the creation of PC games, Project Aela allows a developer to utilise 3D rendering, 2D rendering, controls, LUA scripting, audio and resource management in their applications.

How does a developer use Project Aela?

Using Project Aela in a desktop application is not very complex. Project Aela is a static library, meaning that it can be included and utilised inside of another software application. Here is a very simple piece of software that utilises Project Aela: 

#include "Aela_Engine.h"

int main(int argc, char *args[]) {
    Aela::start();
    return 0;
}

Where did the name "Project Aela" come from?

Project Aela was named after a huntress from a certain video game.
